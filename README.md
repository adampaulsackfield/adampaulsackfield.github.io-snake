# Snake

A Vanilla HTML, CSS and JavaScript implementation of the classic game Snake.
Features

- [✅] Keyboard Controls
- [✅] Smooth Movement
- [❌] Functioning version of the game
- [❌] Ability to see high scores
- [❌] Control buttons for restarting and quitting
- [❌] Fully Responsive
